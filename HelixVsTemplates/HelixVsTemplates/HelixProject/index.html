Getting Started

Sitecore Helix Template

Things to Consider

This project creates the scaffolding for a new Sitecore Helix Peoject.

Keep in mind

  1. You have to manually add the references to the other project dependencies.
  2. You have to add the repository calls.
  3. You have to configure IOC and Dependency Injection.
  4. For the Master TDS project, the Source Web Project has already been set to the "$safeprojectname$" project, but the sitecore instance URI and physical location should be set using the TdsGlobal.config file at solution level.
